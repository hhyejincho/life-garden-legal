매일텃밭 계정 및 데이터 삭제 안내

시행일: 2026년 4월 7일

1. 앱 내에서 직접 삭제

매일텃밭 앱을 사용 중인 경우 다음 경로로 즉시 계정 및 모든 데이터를 삭제할 수 있습니다.

  앱 실행 → 프로필 탭 → 회원탈퇴

탈퇴 즉시 모든 데이터가 영구 삭제되며, 별도의 보관 기간은 없습니다.
(단, 관련 법령에 따라 일부 결제 기록은 일정 기간 보관됩니다. 아래 4항 참고.)

2. 이메일을 통한 삭제 요청

앱을 사용할 수 없거나 앱을 이미 삭제한 경우, 다음 이메일로 삭제 요청을 보내실 수 있습니다.

  연락처: lifegarden.app@gmail.com
  요청 시 포함할 정보:
    • 가입 이메일 주소
    • 닉네임 (선택)
    • 요청 사유 (선택)

영업일 기준 7일 이내 처리되며, 처리 완료 후 안내 메일을 발송합니다.

3. 삭제되는 데이터

회원 탈퇴 또는 삭제 요청 시 다음 데이터가 모두 영구 삭제됩니다.

  • 계정 정보: 이메일 주소, 닉네임, 상태메시지
  • 텃밭 데이터: 식물, 진행도, 좌표, 코인, 창고(수확물)
  • 하루씨앗: 진행 상태, 이력
  • 생각나무 글
  • 활동 뱃지, 농부 캐릭터 꾸미기 데이터
  • 시간표: 루틴 시간표, 하루 일과 시간표
  • 기념일 스티커
  • 친구 관계, 응원 기록, 방명록 (보낸/받은)
  • 동기화 메타데이터 (마지막 수정 시각 등)
  • Supabase Auth 계정 정보

4. 보관되는 데이터 (법령에 의한 의무 보관)

다음 정보는 관련 법령에 따라 일정 기간 보관됩니다.

  • 전자상거래법에 의한 계약/청약철회 기록: 5년
  • 전자상거래법에 의한 소비자 불만/분쟁 기록: 3년

위 정보는 식별 가능한 형태로 보관되지 않으며, 결제 처리 및 분쟁 해결 목적 외에는 활용되지 않습니다.

5. 위탁 업체에서의 삭제

탈퇴 시 다음 위탁 업체의 데이터도 함께 처리됩니다.

  • Supabase Inc. (클라우드 데이터): 즉시 삭제
  • Sentry (오류 진단 정보): 90일 후 자동 만료
  • RevenueCat (결제 영수증): 익명화된 식별자만 보관, 개인 식별 정보 없음
  • Google AdMob (광고식별자): 기기 설정에서 언제든 재설정 가능

자세한 위탁 사항은 개인정보처리방침 5항을 참고하세요.
